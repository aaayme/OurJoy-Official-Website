OurJoy - Development Plan

Phase 1: Core Foundation (Weeks 1–2)

Goals:

  • Set up environment and workflow
  • Get the basic game loop working
  • Test isolated systems

Tasks:

  • Create a clean base place in Roblox Studio
  • Set up folders/modules
  • Set up basic UI system
  • Implement room randomizer prototype
  • 1st/3rd person camera toggle

Phase 2: Core Systems (Weeks 3–4)

Red/Blue Piles System

  • Pile spawning logic
  • Trigger + cooldown
  • Red/blue pile models
  • Animations: eating, effect, GUI shake

Combat System

  • Dash, block, attack
  • Weapon switching and guns
  • 3rd-person view in combat
  • Weapon models
  • Combat animations

Phase 3: Room & Puzzle Systems (Weeks 5–6)

Dynamic Rooms

  • 100 room layout preserving unique rooms
  • Locked doors + keys
  • At least 10 room models

Within Room System

  • Hiding spots, closet/item interactions
  • Work paper & currency logic
  • Room decoration

Puzzle System

  • Basic puzzle logic: switches, generators
  • Models: levers, generator
  • Animations: lever toggles, explosions

Phase 4: Mob & AI System (Weeks 7–8)

Mob System

  • Pathfinding
  • Chase/attack/jump scare logic
  • Mob modeling and animations

Phase 5: Shop & Progression (Week 9)

Shop System

  • GUI with red/blue sections
  • Buy/equip abilities
  • UI polish

Phase 6: Polish & Testing (Weeks 10–11)

  • Final polish (FX, lighting)
  • Multiplayer testing (1–8 players)
  • Difficulty and performance balancing

Phase 7: Launch (Week 12)

  • Launch trailer (optional)
  • Public release
  • Patch and plan updates
